« jQuery Alert Dialogs (Alert, Confirm, & Prompt代替方案)

基本范例

Alert 
jAlert('自定义对话框', 'Alert对话框');

Confirm 
jConfirm('您确定吗?', '确认对话框', function(r) {
    jAlert('Confirmed: ' + r, 'Confirmation Results');
});

Prompt 
jPrompt('请输入keleyi.com或者其他:', 'keleyi.com(预填值)', 'Prompt对话框', function(r) {
    if( r ) alert('You entered ' + r);
});

高级范例

对话框带HTML代码 
jAlert('可以在这里使用HTML, 例如柯乐义、bold, italics, 和 underline!');
